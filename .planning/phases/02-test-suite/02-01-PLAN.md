**startBroker()** - Creates an aedes broker instance, starts a net server on a random available port (use `server.listen(0)` to get a random port), returns `{ broker, server, port, url }`. The `url` is `mqtt://localhost:${port}`. Include a `close()` method that destroys the broker and closes the server.

**spawnSimulator({ brokerUrl, scenario, gateway, device, extraArgs })** - Spawns `node simulator.mjs` as a child process using `child_process.spawn` with the provided args. Default gateway: `TEST-GW-001`, default device: `TEST-DEV-001`. Returns the child process. Set stdio to `['ignore', 'pipe', 'pipe']` so stdout/stderr are capturable but stdin is ignored.

**collectMessages({ brokerUrl, topics, until, timeoutMs })** - Connects an mqtt client to the broker, subscribes to the given topic patterns (e.g. `sys/product/+/status`, `thing/product/+/osd`), collects all received messages as `{ topic, payload }` (payload JSON-parsed). The `until` parameter is a function `(messages) => boolean` that returns true when collection should stop. `timeoutMs` defaults to 15000. Returns the collected messages array. Cleans up the mqtt client on completion.

All three utilities must handle cleanup properly to avoid test hangs.
